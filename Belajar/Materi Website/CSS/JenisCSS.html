Jenis CSS


Jenis CSS ada 3 :

  • Inline
  • Embed
  • Eksternal

Inline

Menyisipkan css pada struktur html seperti berikut
Contoh:

"Dibalik kegagalan pasti ada jalan kesuksesan"

Source Code 

        <p style=" font-family: monospace;
                   font-size: 18px;
                   font-weight: bold;
                   color: red;
                   font-style: italic;">"Dibalik kegagalan pasti ada jalan kesuksesan" 
        </p>

Embed

CSS ditempatkan didalam tag head
Contoh:

"Bersusah-susah dahulu bersenang-senang kemudian"

Source Code

        <head>
            <title>Embed</title>
            <style>
            h3 {
                font-family: Segoe UI;
                font-size: 18px;
                font-weight: normal;
                color: red;
                font-style: italic;
                text-decoration: underline;
              }
            </style>
        </head>

Eksternal

Jenis ini menggunakan file terpisah dengan ekstensi .css. Dalam menggunakan file .css ini kita bs memanggilnya didalam tag head yang didalamnya terdapat tag link untuk menghubungkan css yang telah dibuat sebelumnya.
Contoh :

Ada udang di balik batu. ada maksud tersembunyi

Source Code

        <head>
            <title>Embed</title>
            <link rel="stylesheet" href="css1.css">
        </head>
